IS.bookmarks.html is the websites database of Illegal Services.
The goal is to update it as much as possible to gather a maximum number of links related to warez (piracy) and cybersecurity.
This file is updated at least once a week in order to always stay updated.

There are currently 3 methods to parse and read this file in a human-readable format:

  • IS Bookmarks website:
With this method, you can navigate the database directly trough internet with using your web browser.
This solution is by far the most user-friendly, fastest and easier.

  • IS.bookmarks.html imported in web browser:
With this method, you can import and navigate the database directly trough your web browser's bookmarks bar.
This solution requiere the user to manually update the database to keep it up-to-date.

  • Illegal Services Windows application:
With this method, you can access the database trough a Windows's application.
This solution can be difficult to use for people not familiar with the Windows's command line interface.
It is also slower to use and boring to install due to Windows Defender incorrectly flagging it as malware.
